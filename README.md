# ADO.NET-Exam

------------
Декстопное приложение для работы с базой данных книжного магазина
Функции приложения:
* Добавление книг
* Редактирование книг
* Удаление книг
* Поиск книг по Названию книги, Атвору и Жанру

Так же есть возможность войдя в свой аккаунт сохранять книги для дальнешей покупки
